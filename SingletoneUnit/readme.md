

# 싱글톤 유닛

## 개요

  - 한 파일에 하나만 존재하는 static 클래스를 선언합니다. 이 static 클래스는 매크로로 자동적으로 선언하고 하위 함수에서 호출이 가능하도록 하는 하나의 아키텍처를 제작합니다.

## 구성

### 매크로:
  - 할당 매크로: 해당 객체가 존재하는 메모리를 선언하는 부분입니다. 객체의 목적에 따라 위치를 지정 가능합니다.
  - 참조 매크로: 해당 객체를 불러오는 부분입니다. 함수의 객체로 사용이 가능합니다.
  - 사용 종료 매크로: 해당 객체의 사용을 종료하는 부분입니다. NULL로 바꿉니다.

### 데이터:
  - 참조자: 해당 객체의 메모리를 저장하는 객체입니다.

### 함수:
  - 참조자 확인 함수: 참조자가 해제된 스택 영역인지 확인합니다.
