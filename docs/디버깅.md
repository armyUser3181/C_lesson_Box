# C 언어 테스트 환경 사용법

이 저장소는 C 언어 코드를 간단하게 컴파일하고 실행해 볼 수 있도록 구성되어 있습니다.

## 1. 실행 준비

이 환경에서는 GCC가 이미 설치되어 있습니다.

## 2. 코드 작성 위치

- 메인 소스 파일: `main.c`
- 빌드 설정 파일: `Makefile`

## 3. 실행 방법

### 3-1. 컴파일하기
```bash
make
```

이 명령을 실행하면 `main.c`를 컴파일해서 실행 파일 `main`을 생성합니다.

### 3-2. 실행하기
```bash
make run
```

또는 직접 실행할 수도 있습니다.
```bash
./main
```

## 4. 코드 수정 방법

1. `main.c` 파일을 열어 C 코드를 수정합니다.
2. `make`로 다시 컴파일합니다.
3. `./main` 또는 `make run`으로 결과를 확인합니다.

## 5. 정리하기

```bash
make clean
```

이 명령을 실행하면 생성된 실행 파일 `main`을 삭제합니다.

## 6. 예시

현재 `main.c`에는 아래와 같은 코드가 들어 있습니다.

```c
#include <stdio.h>

int main(void) {
    printf("hello world");
    return 0;
}
```

실행하면 다음처럼 출력됩니다.

```text
hello world
```

## 7. 참고

- C 코드를 작성할 때는 `main.c` 파일을 수정하면 됩니다.
- 문제가 생기면 먼저 `make`를 다시 실행해 보세요.
- 컴파일 오류가 나오면 오류 메시지를 확인해서 코드의 문법을 수정하면 됩니다.
